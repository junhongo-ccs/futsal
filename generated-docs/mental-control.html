からだ、ことば、いしきで整える

核心

結果は直接コントロールしにくい。

ゴールが決まるか、勝てるか、ミスをしないか、味方にどう見られるかは、自分だけでは決められない。

だから、結果を最初に考えすぎると、メンタルが揺れやすくなる。

結果を出し続ける人は、まず自分がコントロールしやすいものを整える。

それが、

1. からだ

2. ことば

3. いしき

の3つ。

この3つが整うと、心の状態が整いやすくなり、パフォーマンスにつながる。

順番

結果を出しにくい流れ:

結果を考えすぎる -> メンタルが揺れる -> パフォーマンスが落ちる

結果を出しやすい流れ:

からだ、ことば、いしきを整える -> メンタルが整う -> パフォーマンスが出やすくなる -> 結果につながる

結果を無視するのではない。

結果を直接つかみに行くのではなく、結果につながる状態を先につくる。

からだ

からだは、姿勢、呼吸、目線、動きのこと。

心を言葉で変えようとしても難しいときは、先に身体を変える。

例:

  • 胸を開く
  • 目線を上げる
  • 呼吸を吐く
  • 肩の力を抜く
  • 最初の一歩を出す
  • ミス後の最初の3歩を早くする
  • 足を止めない

本郷さんにとっては、

観る。探る。決断。体現。

が、状況を見てから身体で実行するまでの一連の合図になる。

ことば

ことばは、自分にかける短い言葉。

長い反省や分析ではなく、プレー中に身体が動く言葉にする。

例:

  • 失敗してもいい
  • 次に関わる
  • 顔を上げる
  • 半歩前へ
  • ミス後3歩
  • 見てから動く
  • もったいない場面は入口

ことばは、メンタルを立て直す合図になる。

「失敗したらどうしよう」ではなく、

失敗してもいい。次に関わる。

に変える。

「あそこはもったいなかった」と言われたら、

もったいない場面は入口。次はトライする。

に変える。

いしき

いしきは、何を見るか、何へ注意を向けるか。

不安や結果に意識が向くと、視野が狭くなりやすい。

意識を外へ向ける。

例:

  • 味方の動き
  • ゴール前のスペース
  • パスコース
  • 守備に戻る方向
  • 次に関わる位置
  • ミス後の切り替え

本郷さんにとっては、

見る、切り替える、次に関わる。

が意識の軸になる。

フットサル前の使い方

試合前に、結果ではなく、この3つを決める。

からだ:

顔を上げる。半歩前へ。

ことば:

失敗してもいい。次に関わる。

いしき:

ゴール前に入る味方を見る。

その上で、技術として何をトライするかを決める。

技術:

アシストになるラストパスを1本狙う。

点数をつけて試す

からだ、ことば、いしきで整えるときは、変化を主観的に点数化するとわかりやすい。

手順:

1. 今の身体と心に0から10で点数をつける

2. 身体または心を1点上げるために、できそうなことをいくつか挙げる

3. その中から1つ試す

4. 試した後に、もう一度点数をつける

例:

最初:

身体 4、心 5。

試すこと:

顔を洗う。音楽を聴く。コーヒーを飲む。少し歩く。

試した後:

身体 4、心 6。

点数が1点でも上がれば、自分で状態を動かせた感覚が残る。

上がらなくても失敗ではない。何をすると変わりやすいか、何をしても変わりにくいかがわかる。

口出しをしない

点数は本人の主観でつける。

周りの人が「それは低すぎる」「もっと高いはず」と口出しすると、本人が自分の状態を感じ取る練習になりにくい。

自分で感じ、自分で点数をつけ、自分で試し、もう一度自分で確認する。

これは、他人の正解に合わせるためではなく、自分の身体と心を自分で扱えるようになるための練習。

結果を考えすぎたとき

結果を考えること自体は悪くない。

ただし、結果を考えすぎて身体が止まるなら、整える順番に戻る。

1. からだ: 顔を上げる、呼吸を吐く

2. ことば: 失敗してもいい、次に関わる

3. いしき: 味方とスペースを見る

この順番で戻れば、メンタルを立て直しやすい。

振り返る問い

  • 結果を考えすぎて、身体が止まった場面はあったか
  • からだを整える合図は使えたか
  • 自分にかけることばは機能したか
  • 意識を味方、スペース、次のプレーへ向けられたか
  • 試す前と後で、身体や心の点数はどう変わったか
  • 点数は自分の主観でつけられたか
  • 次回は、からだ、ことば、いしきのどれを先に整えるか

立ち戻る言葉

結果ではなく、まず状態を整える。

からだ、ことば、いしき。

顔を上げる。失敗してもいい。次に関わる。
