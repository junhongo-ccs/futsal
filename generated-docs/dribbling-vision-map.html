ドリブル中の視線マップ

核心

ドリブル中に迷いやすいのは、ボール、相手、味方、スペースのどれも大事に見えるから。

そこで、感覚任せではなく、

見る場所、見る順番、見る頻度を決めておく。

これが視線マップ。

視線は、技術と判断をつなぐ導線。

見る場所が整理されると、判断が早くなり、タッチも安定しやすくなる。

三層の視野レイヤー

視線は大きく3つに分ける。

足元

役割は、ボールタッチ直前と直後の確認。

見すぎない。

必要な瞬間だけ、短く見る。

近距離

役割は、1〜5mの相手、味方、ライン、間合いを見ること。

相手の足が出るか、寄せが速いか、パスコースが開いているかを周辺視で拾う。

遠距離

役割は、次のスペース、逆サイド、背後のランナー、ゴール前の状況を先読みすること。

今のタッチではなく、次の目的地を見る。

基本の比率

目安は、

足元 0.5、周辺視 7、遠く 2.5

くらい。

足元を見続けると、相手やスペースの変化に遅れる。

ボールは触覚と周辺視で管理し、焦点視は短く使う。

合図は、

タッチ直前だけ足元。タッチ後は顔を上げる。

見る順番

優先順位は、

危険 → 機会 → 管理

の順。

危険:

  • 誰が寄せてくるか
  • どの方向から来るか
  • 相手の足が届くか
  • カバーがいるか

機会:

  • 空いているスペース
  • 味方のサポート角度
  • 逆サイド
  • 突破後の出口

管理:

  • ボールの位置
  • 次のタッチの強さ
  • 足元のズレ

結果を急ぐより、まず危険を見て、次に機会を見て、最後にボールを管理する。

先行注視

受ける前、仕掛ける前に見ることを先行注視という。

これは、未来の自分に情報を渡す行為。

受ける1〜2秒前に、左右、前、背後を小さく確認する。

先行注視が足りないと、

  • 寄せに驚く
  • タッチが大きくなる
  • 味方の動きを見逃す
  • 空いていたレーンが閉じてから気づく

ということが起きやすい。

体の向き

視線は、体の向きに左右される。

半身を作ると、見える景色が広がる。

  • 受ける前: 足と胸を45度ほど開く
  • 運ぶとき: 外足タッチを使い、相手を視界に入れる
  • 切り返した後: 最初の一歩で半身を作り直す

体の向きは、視線の土台。

見ようとする前に、見える姿勢を作る。

場面別の視線

1対1で仕掛ける前

見る順番:

1. 相手の腰、膝、軸足

2. 背後のカバー

3. 抜いた後の出口

4. 足元は最小限

相手の足先だけを見ると引っかかりやすい。

腰、膝、軸足を見る。

1対1の最中

フェイント前は相手の腰と前足を見る。

踏み込む瞬間だけ足元を見る。

抜き出しの1歩目では、出口とカバーを見る。

前進ドリブル

スペースがあるときは、遠くを見る比率を増やす。

  • 最終ラインの背後
  • 斜めのレーン
  • 寄せてくる相手の初動
  • ロングタッチの方向

加速するほど、先に見る。

密集地帯

密集では、視線を低く固定しない。

小刻みにチラ見する。

  • 受ける前に背後と斜め後ろを確認する
  • 足元はショートタッチで管理する
  • 反転できる窓を探す
  • 顔を上げる時間を短く何度も作る

サイドでの縦突破

見る順番:

1. ライン際のスペース

2. 相手のカバー角度

3. ゴール前、味方の位置

4. 足元

カットイン

見る順番:

1. 内側の相手

2. シュートコース

3. 逆サイドの味方

4. 足元

中央で運ぶ

見るもの:

  • 中盤ラインのズレ
  • 縦パスの窓
  • 味方のサポート角度
  • 寄せのスイッチ役

中央は選択肢が多いので、足元に視線が落ちるほど判断が遅れる。

相手のどこを見るか

相手を見るときは、足先だけを見ない。

  • 腰: 進行方向が出やすい
  • 膝: 踏み出しの方向が出やすい
  • 軸足: タックルの届く範囲がわかる

決定の瞬間は、腰、膝、軸足を見る。

視線サイン

味方との連携では、視線もサインになる。

  • 一瞬見る: 足元で受けたい
  • 裏を見る: 裏へ走ってほしい
  • 味方を見るふりをする: 相手を動かす
  • 肩の向きと視線をそろえる: 意図が伝わりやすい

ただし、長く見すぎると相手にも読まれる。

合図は短く出す。

練習

コーンドリブルと視線タスク

コーンを通りながら、前の人が出す番号や色を声に出す。

足元を見るのはタッチ直前だけ。

目的は、顔を上げる癖と判断の切り替えを作ること。

ミラードリブル

2人組で、相手の腰を見ながら同じ動きを追う。

足先を見るのは禁止。

腰を見る癖を作る。

小スペースロンド

受ける前に左右どちらかを必ず見る。

見た方向と逆へ展開できたら加点。

先行注視を自動化する。

よくあるミス

ボールばかり見る

修正:

  • タッチ前だけ足元を見る
  • タッチ後に必ず顔を上げる
  • 動画で顔が下がった時間を見る

相手の足先だけを見る

修正:

  • 足先ではなく、膝、腰、軸足へ視点を上げる

視線が止まる

修正:

  • 同じ場所を長く見ない
  • 見る、外す、戻すをリズム化する

切り返し後に顔が上がらない

修正:

曲がる前に見る。曲がったらまた見る。

試合前の自分用視線マップ

試合前は3つだけ決める。

1. 誰の寄せが一番速いか

2. どこが空きやすいか

3. 最初にどこを見るか

全部を見ようとしない。

自分の今日の視線ルートを決める。

例:

背後を見る → 前の窓を見る → 寄せ役を見る → 足元を見る。

今日の合図

  • 危険、機会、管理
  • 先に見る
  • タッチ後に顔を上げる
  • 腰、膝、軸足
  • 曲がる前に見る。曲がったらまた見る
