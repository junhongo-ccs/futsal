# 自分のフットサルの目的

## 目的

週末フットサルの目的は、最後まで走り切れる自分でありながら、挑戦したいプレーにトライし、ゴールやアシストで仲間に貢献することで、清々しい達成感と向上心を育てること。

## 短い言葉

> 挑戦し、走り切り、仲間に貢献して、清々しい向上心を持ち帰ること。

## 自分にとっての「来てよかった」

終わったあとに、清々しい気持ちや、やりきった気持ちがあること。

特に大事なのは、トライしてみようと思ったことに実際にトライできたかどうか。

味方からのアシストをうまくゴールに決められたとき、逆に味方にもたくさんアシストパスを出せたとき、「本郷さん、サンキュー」と言われるようなプレーができたとき、来てよかったと思える。

最後の時間まで走って、試合の流れについていけていることも大切。

## 半年後に残っていてほしいもの

向上心。

うまくなりたい、もっとできるようになりたい、次はこうしてみたい、という感覚を持ち続けること。

## 立ち戻る問い

- 今日、自分は何にトライするのか
- 味方にどう貢献したいのか
- 最後まで走るために、どうプレーするのか
- 終わったあと、どんな清々しさを持ち帰りたいのか
- 次回につながる向上心は何か
