# 狭いコートでのプレッシャー対処

## 核心

フットサルはコートが狭く、相手の寄せも速い。

だから、プレッシャーを感じないようにするのではなく、プレッシャーがある状態でも、見る、切り替える、次に関わることが大切。

## 1. プレッシャーや不安を受け入れる

緊張やプレッシャーを無理に下げようとすると、かえって焦りやストレスが強くなることがある。

まずは、

> 今、自分は緊張している。

と認める。

プレッシャーを消そうとするのではなく、あるものとして受け入れる。そのうえで、本来見るべきものや、次にやる行動へ戻る。

## 2. 自分の内部ではなく外部と今に意識を向ける

プレッシャーがかかると、自分の身体の動きやミスへの不安に意識が向きやすい。

これを内部フォーカスに寄りすぎた状態と考える。

内部に意識が向きすぎると、余計な力が入り、判断も遅れやすくなる。

そこで、意識を外へ向ける。

- 相手の位置
- 味方の動き
- ボールを運べるスペース
- パスコース
- 次に関わる位置

過去のミスや未来の不安ではなく、今この瞬間のプレーに戻る。

短く言うと、

> 外を見る。今に戻る。

## 3. 足裏コントロールで顔を上げる

フットサルでは、足裏でボールを扱えると、視線をボールから切り離しやすくなる。

足裏でボールの感触をつかむことで、顔を上げる余裕が生まれる。

顔が上がると、味方、相手、スペースが見えやすくなる。

狭い局面でも、一瞬の間やタメを作りやすくなる。

意識すること:

- 足裏で止める
- 顔を上げる
- 相手の出方を見る
- 味方の動きを見る
- 半歩前へ出るか、横へ逃がすかを選ぶ

短く言うと、

> 足裏、顔上げ、外を見る。

## 4. セルフトークで切り替える

ミスをした直後やプレッシャーを感じたときは、自分にかける言葉を短く決めておく。

長い反省ではなく、次の行動につながる言葉にする。

例:

- 失敗してもいい。次に関わる。
- もったいない場面は入口。
- 顔を上げる。
- 半歩前へ。
- 今に戻る。
- 外を見る。
- 楽しんでいこう。

「失敗したらどうしよう」ではなく、

> 失敗しても何か得られる。

に変える。

「緊張する」ではなく、

> 今、集中できる状態に入っている。

に変える。

## 5. プレッシャーへの免疫をつける

普段から、少しだけプレッシャーがかかる課題を自分に出す。

小さな成功体験を積み重ねることで、本番で過度に緊張しても戻りやすくなる。

例:

- 今日は1回、迷ったらパスを出す
- 今日は1回、足裏で止めて顔を上げる
- 今日はミス後3歩だけ早く戻る
- 今日は「もったいない」と思う場面で1回トライする

評価は結果ではなく、行動で見る。

> 自分で決めた約束を行動に移せたか。

これをインサイドルールとして持つ。

## 本郷さん用の実戦セット

狭い局面でプレッシャーを感じたら、まずはこの順番で戻る。

1. 受け入れる: 今、緊張している
2. からだ: 足裏で止める、顔を上げる
3. いしき: 外を見る、味方とスペースを見る
4. ことば: 失敗してもいい、次に関わる
5. 行動: 半歩前へ、または逃がす

短い言葉:

> 足裏、顔上げ、外を見る。

> 失敗してもいい。次に関わる。

> 今に戻る。半歩前へ。

## 振り返る問い

- プレッシャーを消そうとして焦った場面はあったか
- 緊張している自分を受け入れられたか
- 顔を上げて外を見られたか
- 足裏で一瞬の余裕を作れたか
- ミス後にセルフトークで切り替えられたか
- 自分で決めた約束を行動に移せたか
